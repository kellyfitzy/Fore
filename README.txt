Need Go 1.5.1, go_appengine 1.9.27, Github

Web locations:
https://appengine.google.com/
http://merigolfer.appspot.com/

Steps to make changes to Fore golf program
1. Run GitHub to get latest code
	- can't be connected to Intel or clone/synch won't work
2. Edit code in /Users/kellyfitzpatrick/fore (~/fore)
        - Use VI, not Apple text editor, or bad characters get inserted
3. Locally test by
	- ./make_fore
	- enter in Chrome:  http://localhost:8080/
4. Deploy to appengine with
	- ./deploy_fore
5. Commit code changes back with GitHub
